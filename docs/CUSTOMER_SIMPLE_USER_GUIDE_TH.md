# คู่มือใช้งานระบบคัดแยกขยะอัตโนมัติ สำหรับลูกค้า

เอกสารนี้เป็นคู่มือใช้งานแบบย่อสำหรับผู้ใช้งานเครื่องคัดแยกขยะอัตโนมัติ หลังจากติดตั้งระบบและตั้งค่าเครื่องเรียบร้อยแล้ว จุดประสงค์คือให้ผู้ใช้งานสามารถเปิดโปรแกรม ใช้งานหน้า UI ดูสถานะเครื่อง และหยุดระบบได้อย่างถูกต้อง

---

## 1. สิ่งที่ต้องตรวจสอบก่อนเปิดโปรแกรม

ก่อนเริ่มใช้งานทุกครั้ง ให้ตรวจสอบรายการต่อไปนี้

- เครื่อง Jetson หรือคอมพิวเตอร์หลักเปิดอยู่
- กล้อง RealSense ต่ออยู่กับเครื่อง
- แขนกลและ slider ต่อสายสัญญาณเรียบร้อย
- สายพานพร้อมใช้งาน และไม่มีสิ่งกีดขวาง
- หัวดูดหรือ vacuum อยู่ในสภาพพร้อมใช้งาน
- ไม่มีมือหรือวัตถุอื่นอยู่ในพื้นที่ทำงานของแขนกล
- โฟลเดอร์โปรแกรมอยู่ที่ path ที่กำหนด เช่น `/home/lab/AI-Vision-Wave`

---

## 2. วิธีเปิดโปรแกรมหลัก

เปิด Terminal แล้วรันคำสั่งต่อไปนี้

```bash
cd /home/lab/AI-Vision-Wave
./scripts/run_machine.sh
```

หลังรันคำสั่ง โปรแกรมหลักจะแสดงหน้าต่าง UI ของระบบคัดแยก

สิ่งที่ควรเห็นหลังเปิดโปรแกรม:

- มีหน้าต่างโปรแกรม `WASTE SORTING SYSTEM PROTOTYPE VERSION`
- ด้านบนมีสถานะ `Camera`, `Robot`, `Spectrum`
- มีแท็บหลัก เช่น `Vision`, `Robot Control`, `Spectrum Scan`, `Video & Recording`, `Calibration`, `Dashboard`

---

## 3. ลำดับการเริ่มใช้งานประจำวัน

แนะนำให้เริ่มใช้งานตามลำดับนี้

1. เปิดโปรแกรมด้วย `./scripts/run_machine.sh`
2. ดูสถานะด้านบนของโปรแกรม
3. ไปที่แท็บ `Vision`
4. กด `CONNECT` เพื่อเชื่อมต่อแขนกลและ slider
5. กด `HOME` เพื่อให้แขนกลกลับตำแหน่งเริ่มต้น
6. กด `START` เพื่อเปิดกล้องและเริ่มตรวจจับวัตถุ
7. ตรวจดูภาพจาก `Vision Feed`
8. เมื่อระบบพร้อม ให้กด `AUTO PICK`
9. เริ่มวางวัตถุบนสายพานตามขั้นตอนปฏิบัติงาน

หมายเหตุ: หากยังไม่แน่ใจว่าสถานะเครื่องพร้อมหรือไม่ ให้ดู `System Log` ด้านล่างซ้ายของแท็บ `Vision`

## 4. คำอธิบายหน้า UI หลัก

## 4.1 แถบสถานะด้านบน

ด้านบนของโปรแกรมจะแสดงสถานะสำคัญของระบบ

| สถานะ | ความหมาย |
| --- | --- |
| `Camera` | สถานะกล้องและภาพจากระบบ vision |
| `Robot` | สถานะแขนกลและ slider |
| `Spectrum` | สถานะ sensor สำหรับอ่านค่าวัสดุ |

สีหรือเครื่องหมายสถานะช่วยบอกว่าระบบพร้อมหรือยัง หากมีสถานะผิดปกติ ให้ตรวจสอบสายเชื่อมต่อและดูข้อความใน `System Log`

---

## 4.2 แท็บ Vision

แท็บ `Vision` เป็นหน้าหลักที่ผู้ใช้งานใช้บ่อยที่สุด

ส่วนสำคัญในแท็บนี้:

| ส่วนของหน้าจอ | ใช้ทำอะไร |
| --- | --- |
| `Vision Feed` | แสดงภาพจากกล้องและวัตถุที่ตรวจจับได้ |
| `Tracking Dashboard` | แสดงรายการวัตถุที่ระบบกำลังติดตาม เช่น ID, Class, Height, X, Y, Status |
| `System Log` | แสดงข้อความสถานะและ error ของระบบ |
| `Main Controls` | ปุ่มควบคุมหลัก เช่น START, STOP, AUTO PICK |
| `Robot Connection` | เชื่อมต่อแขนกล, slider และสั่ง HOME |
| `System Parameters` | ปรับ model, belt speed, confidence และค่าที่เกี่ยวข้อง |
| `Sorting Statistics` | แสดงจำนวนวัตถุที่คัดแยกตามประเภท |

ปุ่มหลักในแท็บ `Vision`:

| ปุ่ม | วิธีใช้ |
| --- | --- |
| `START` | เริ่มกล้องและระบบตรวจจับ |
| `STOP` | หยุดระบบตรวจจับและการทำงานหลัก |
| `AUTO PICK` | เปิด/ปิดโหมดหยิบและคัดแยกอัตโนมัติ |
| `CONNECT` | เชื่อมต่อแขนกลและ slider |
| `DISCONNECT` | ตัดการเชื่อมต่อแขนกล |
| `HOME` | สั่งแขนกลกลับตำแหน่งเริ่มต้น |
| `RELOAD CAL` | โหลดค่า calibration ใหม่จากไฟล์ |

การใช้งานปกติ:

```text
CONNECT -> HOME -> START -> AUTO PICK
```

## 4.3 แท็บ Robot Control

แท็บ `Robot Control` ใช้สำหรับตรวจสอบและควบคุมแขนกลแบบ manual

ผู้ใช้งานทั่วไปไม่ควรปรับค่าในแท็บนี้หากไม่ได้รับคำแนะนำจากผู้ดูแลระบบ

ส่วนที่พบในแท็บนี้:

| ส่วนของหน้าจอ | ใช้ทำอะไร |
| --- | --- |
| `Robot Camera` | แสดงภาพจากกล้องที่ติดกับแขนกล |
| `Current Position` | แสดงตำแหน่ง X, Y, Z, R ของแขนกล |
| `Manual Jog Control` | ขยับแขนกลทีละแกน |
| `Direct Position Control` | สั่งแขนกลไปตำแหน่งที่กำหนด |
| `Vacuum Control` | เปิด/ปิดหัวดูด |
| `Place Positions` | ตั้งค่าตำแหน่งปล่อยวัตถุแต่ละประเภท |

ข้อควรระวัง:

- ห้ามกด jog หรือ direct position หากมีคนอยู่ใกล้พื้นที่แขนกล
- ห้ามเปลี่ยนตำแหน่ง `Place Positions` โดยไม่บันทึกค่าหรือไม่ได้รับอนุญาต
- ถ้าต้องทดสอบ vacuum ให้ตรวจสอบพื้นที่ทำงานก่อนทุกครั้ง

---

## 4.4 แท็บ Spectrum Scan

แท็บ `Spectrum Scan` ใช้สำหรับทดสอบ sensor ตรวจวัดวัสดุด้วย spectrum

ส่วนสำคัญ:

| ส่วนของหน้าจอ | ใช้ทำอะไร |
| --- | --- |
| `Sensor Status` | แสดงว่าสถานะ sensor พร้อมหรือไม่ |
| `SCAN NOW` | สั่งอ่านค่า spectrum ทันที |
| `Last Prediction` | แสดงผลทำนายวัสดุล่าสุด |
| `Confidence` | แสดงความมั่นใจของผลทำนาย |
| `Raw Spectrum Data` | แสดงค่าดิบจาก sensor |

ผู้ใช้งานทั่วไปใช้แท็บนี้เมื่อต้องตรวจสอบว่า spectrum sensor ยังอ่านค่าได้ปกติ

---

## 4.5 แท็บ Video & Recording

แท็บ `Video & Recording` ใช้สำหรับเล่นไฟล์ย้อนหลังหรือบันทึกข้อมูลการทดลอง

ส่วนสำคัญ:

| ส่วนของหน้าจอ | ใช้ทำอะไร |
| --- | --- |
| `LOAD .bag/.mp4` | โหลดไฟล์วิดีโอหรือไฟล์บันทึก |
| `PLAY` | เล่นไฟล์ที่โหลด |
| `STOP` | หยุดเล่นไฟล์ |
| `REC .bag` | บันทึกข้อมูล RealSense แบบ `.bag` |
| `REC Experiment` | บันทึกวิดีโอการทดลองและข้อมูลประกอบ |
| `Start Audit Log` | เก็บข้อมูลสำหรับตรวจสอบผลการคัดแยก |

ผู้ใช้งานทั่วไปไม่จำเป็นต้องใช้แท็บนี้ในการทำงานประจำวัน ยกเว้นกรณีที่ต้องเก็บวิดีโอหรือข้อมูลให้ทีม support ตรวจสอบ

---

## 4.6 แท็บ Calibration

แท็บ `Calibration` ใช้สำหรับตั้งค่าพื้นที่สายพานและค่าความลึกของพื้นสายพาน

ควรใช้แท็บนี้เมื่อ:

- มีการขยับตำแหน่งกล้อง
- เปลี่ยนตำแหน่งสายพาน
- ภาพตรวจจับไม่ตรงกับตำแหน่งจริง
- ความสูงวัตถุผิดปกติ
- ทีม support แจ้งให้ calibrate ใหม่

ขั้นตอนโดยย่อ:

1. กด `START` ให้กล้องทำงานก่อน
2. วาง checkerboard บนสายพาน
3. ไปที่แท็บ `Calibration`
4. กด `ENABLE CALIBRATION`
5. ตรวจแนว ROI บนภาพ
6. กด `CONFIRM CALIBRATION`
7. เอาวัตถุออกจากสายพาน
8. กด `CALIBRATE FLOOR`
9. กด `SAVE CALIBRATION`

หมายเหตุ: การ calibration ควรทำโดยผู้ดูแลระบบหรือ technician

| `Session Log` | รายการ log ในแต่ละ session |

## 5. วิธีหยุดระบบ

เมื่อต้องการหยุดใช้งาน ให้ทำตามลำดับนี้

1. กด `AUTO PICK` เพื่อปิดโหมดหยิบอัตโนมัติ หากเปิดอยู่
2. รอให้แขนกลจบรอบการทำงานปัจจุบัน
3. กด `STOP`
4. กด `DISCONNECT` หากต้องการตัดการเชื่อมต่อแขนกล
5. ปิดหน้าต่างโปรแกรม
6. ปิดเครื่องตามขั้นตอนของหน้างาน

ข้อควรระวัง:

- ห้ามปิดเครื่องทันทีขณะแขนกลกำลังหยิบวัตถุ
- หากเกิดเหตุฉุกเฉิน ให้ใช้ปุ่มหยุดฉุกเฉินของเครื่องก่อน

## 6. การเปิดเว็บ Dashboard กลาง

ถ้ามีการติดตั้ง dashboard กลาง ให้เปิดผ่าน browser

```text
http://<server-ip>:5000
```

ข้อมูลบน dashboard กลางมาจากแต่ละเครื่องที่ส่งข้อมูลทุก 1 นาทีผ่าน sync worker

ถ้า dashboard ไม่มีข้อมูล ให้ตรวจสอบว่า sync worker ทำงานอยู่:

```bash
cd /home/lab/AI-Vision-Wave
./scripts/run_sync_worker.sh --once
```

## 7. ปัญหาที่พบบ่อย

| ปัญหา | วิธีตรวจสอบเบื้องต้น |
| --- | --- |
| โปรแกรมเปิดไม่ได้ | ตรวจ path และรัน `./scripts/run_machine.sh` จาก root ของโปรเจค |
| กล้องไม่ขึ้นภาพ | ตรวจสาย USB ของ RealSense และกด `START` ใหม่ |
| Robot ไม่เชื่อมต่อ | ตรวจ port, สาย USB/serial และกด `CONNECT` ใหม่ |
| กด AUTO PICK แล้วไม่หยิบ | ตรวจว่า `START` ทำงานอยู่, robot connected และมีวัตถุในภาพ |
| Class ผิดบ่อย | ตรวจ model, calibration และ spectrum sensor |
| ตำแหน่งหยิบคลาด | ตรวจ `offsets.json` หรือทำ calibration ใหม่ |
| Dashboard ไม่มีข้อมูล | ตรวจ sync worker, network และค่า `DASHBOARD_URL` |
| script รันไม่ได้ | ใช้ `chmod +x scripts/*.sh` |

## 9. สรุปการใช้งานแบบสั้นที่สุด

ลำดับการใช้งานปกติ:

```text
เปิดเครื่อง -> เปิดโปรแกรม -> CONNECT -> HOME -> START -> AUTO PICK -> ใช้งาน
```

ลำดับการหยุดระบบ:

```text
ปิด AUTO PICK -> รอแขนกลจบรอบ -> STOP -> DISCONNECT -> ปิดโปรแกรม
```

เอกสารนี้ตั้งใจให้ลูกค้าสามารถเปิดระบบและใช้งานหน้า UI ได้ทันที โดยไม่จำเป็นต้องเข้าใจรายละเอียดเชิงเทคนิคทั้งหมดของระบบ
